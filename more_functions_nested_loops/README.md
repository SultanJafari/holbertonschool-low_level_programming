# C - More functions, more nested loops

This project is part of the Software Engineering curriculum at Holberton School. It focuses on mastering functions, nested loops, and header files in the C programming language.

## Learning Objectives
- Understanding nested loops and their applications.
- How to define and use functions.
- Difference between function declaration (prototype) and definition.
- Understanding scope of variables in C.
- How to use header files to organize code.
- Familiarity with GCC flags (`-Wall`, `-Werror`, `-Wextra`, `-pedantic`, `-std=gnu89`).

## Requirements
- Allowed editors: `vi`, `vim`, `emacs`.
- All files are compiled on Ubuntu 20.04 LTS using `gcc`.
- All files must end with a new line.
- Code must follow the **Betty style**.
- No global variables allowed.
- No standard library functions (like `printf`, `puts`) are allowed, only `_putchar`.
