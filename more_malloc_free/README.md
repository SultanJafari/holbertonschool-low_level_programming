# C - More malloc, free

This project is part of the low-level programming and algorithm curriculum at Holberton School. It focuses on advanced memory allocation and management using `malloc`, `free`, `calloc`, and `realloc` in C.

## Tasks

* **0. Trust no one** (`0-malloc_checked.c`): Function that allocates memory using `malloc` and exits with status `98` if it fails.
* **1. string_nconcat** (`1-string_nconcat.c`): Function that concatenates two strings up to `n` bytes.
* **2. _calloc** (`2-calloc.c`): Function that allocates memory for an array, using `malloc`.
* **3. array_range** (`3-array_range.c`): Function that creates an array of integers.
