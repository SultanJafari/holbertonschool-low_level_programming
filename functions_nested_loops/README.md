# Functions and nested loops

This project is a collection of C programs that explore the use of functions, prototypes, header files, and nested loops in the C programming language.

## Learning Objectives
- Understanding nested loops and how to use them.
- Creating and using functions in C.
- Understanding the difference between a declaration and a definition.
- Working with header files and `#include`.
- Understanding scope of variables.
- Using GCC flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`.

## Requirements
- All files are compiled on Ubuntu 20.04 LTS using `gcc`.
- All code follows the Betty coding style.
- No global variables are allowed.
- The use of standard library functions like `printf` or `puts` is forbidden.
- `_putchar` is used for output.
