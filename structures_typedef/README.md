# C - Structures, typedef

This project is part of the Holberton School low-level programming curriculum. It covers the fundamentals of structures in the C programming language, how to define and use them, and how to utilize `typedef` to simplify code.

## Resources
* Structures
* Typedef and structures
* Programming in C by Stephen Kochan - Chapter 8

## Requirements
* Allowed editors: `vi`, `vim`, `emacs`
* Compiled on Ubuntu 20.04 LTS using `gcc`, using the options: `-Wall -Werror -Wextra -pedantic -std=gnu89`
* All files should end with a new line
* Code must follow the **Betty style**
* No global variables allowed
* No more than 5 functions per file
* The only allowed standard library functions are `printf`, `malloc`, `free`, and `exit`.
* All header files must be include-guarded.

## Tasks
* **0. My first dog** (`0-struct_dog.c`, `dog.h`): Define a new type `struct dog` with elements `name`, `age`, and `owner`.
* **1. Initialize a variable** (`1-init_dog.c`): Function that initializes a variable of type `struct dog`.
* **2. Print a dog** (`2-print_dog.c`): Function that prints a `struct dog`.
* **3. New dog** (`3-new_dog.c`): Function that creates a new dog.
* **4. Now, we free these dogs** (`4-free_dog.c`): Function that frees dogs.
